# End Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 抽奖全部结束后跳转到永久停留的结束页，展示"THANK YOU"大字幕和领奖提示，替代原来回到 HOME 的逻辑。

**Architecture:** 新增 `END` 状态到现有状态机，修改 `nextRound()` 在最后一轮后进入 `END` 而非重置；新建 `EndView` 组件复用 `MainLayout` 外壳；`App.jsx` 增加 END 分支渲染。

**Tech Stack:** React 18, Framer Motion, Tailwind CSS, Playwright（E2E 测试）

### Task 2: 在 LotteryContext 中新增 END 状态

**Files:**
- Modify: `src/store/LotteryContext.jsx`

- [ ] **Step 1: 修改 `nextRound()` — 最后一轮进入 END**

找到 `nextRound` 函数（当前第 56-67 行），替换为：

```javascript
const nextRound = () => {
    if (currentRoundIndex < ROUNDS.length - 1) {
        setCurrentRoundIndex(prev => prev + 1);
        setCurrentPhase('WAIT');
    } else {
        setCurrentPhase('END');
    }
};
```

> 移除原来的 `resetLottery()` 调用，结束后不再重置。

- [ ] **Step 2: 更新键盘监听 — END 状态不响应按键**

找到 `handleKeyDown` 内的条件判断（当前第 108-112 行），在末尾确认没有 `END` 分支即可——原代码不匹配 `END`，已天然忽略。

验证当前键盘监听代码：

```javascript
if (currentPhase === 'HOME') startEvent();
else if (currentPhase === 'WAIT') startDrawing();
else if (currentPhase === 'DRAWING') stopDrawing();
else if (currentPhase === 'RESULT') nextRound();
// END 状态：无分支，Space/Enter 不执行任何操作 ✓
```

无需额外修改。

- [ ] **Step 3: 确认 context value 无需新增字段**

`value` 对象不需要暴露新方法，END 状态由 `currentPhase` 驱动渲染，消费方通过 `useLottery()` 读取即可。

### Task 4: 在 App.jsx 中接入 EndView

**Files:**
- Modify: `src/App.jsx`

- [ ] **Step 1: 导入 EndView 并添加 END 分支**

将 `src/App.jsx` 完整替换为：

```jsx
import React from 'react';
import MainLayout from './components/Layout/MainLayout';
import HomeView from './views/HomeView';
import LotteryDisplay from './views/LotteryDisplay';
import EndView from './views/EndView';
import { useLottery } from './store/LotteryContext';

function App() {
    const { currentPhase } = useLottery();

    return (
        <MainLayout>
            {currentPhase === 'HOME' && <HomeView />}
            {(currentPhase === 'WAIT' || currentPhase === 'DRAWING' || currentPhase === 'RESULT') && (
                <LotteryDisplay />
            )}
            {currentPhase === 'END' && <EndView />}
        </MainLayout>
    );
}

export default App;
```
